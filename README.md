# 🎬 Movie Discovery Web App

🚀 A modern movie discovery application built with **React.js, Tailwind CSS, and the OMDb API**, allowing users to search and explore movies in real time.

---

## ✨ Features

- 🔍 Real-time movie search using OMDb API  
- ⭐ IMDb ratings and detailed movie information

<p align="center">
  <img src="https://github.com/TonimaDristy/ReactJS/raw/main/307583e4-14ac-46b9-a4c1-889d452389e2.jpg" width="50%" />

- 📅 Release date display  
- 🎨 Clean and responsive UI design  
- ⏳ Loading states for better user experience  
- ❌ Error handling for invalid searches  
- 🖼️ Fallback UI (shows “Poster Not Available” when image is missing)  

---

## 🛠️ Tech Stack

- React.js  
- Tailwind CSS  
- OMDb API  
- JavaScript (ES6+)  

---
